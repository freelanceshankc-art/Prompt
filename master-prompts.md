# Master Prompts for Facebook Reels

A complete reference of text-to-image (T2I) and image-to-video (I2V) prompts organized by niche. Designed for **9:16 vertical format**, optimized for Facebook Reels, Instagram Reels, TikTok, and YouTube Shorts.

---

## Universal Formula

**Text-to-Image:**
```
[Subject] + [Action] + [Environment] + [Style] + [Lighting] + [Camera] + [9:16 specs]
```

**Image-to-Video:**
```
[Camera move] + [Subject motion] + [Environmental motion] + [Duration] + [Mood]
```

**Always append to T2I prompts:**
> 9:16 vertical aspect ratio, subject in lower-third, clean negative space at top for text overlay, hyper-detailed, 8K, cinematic color grading, trending social media aesthetic

## Pro Tips for Every Niche

1. **Hook frame** — design first frame to stop the scroll
2. **Leave 20% headroom** at top for caption text
3. **High-contrast subjects** for mobile readability
4. **Keep I2V prompts simple** — too many motions cause AI artifacts
5. **Trigger words**: cinematic, hyper-realistic, viral, trending, dynamic
6. **Avoid** in I2V: complex multi-subject scenes, fast cuts, text rendering
7. **Test sound** — Reels with trending audio outperform silent ones
